# Advertisement-Click-Through-Rate
Repo for a model that analyzes the effectiveness of online advertisements by measuring the rate at which users click on an ad’s link to reach the advertiser’s website.

# Overview
* The repo provides an output of forecasts for the click-through rate(CTR). It is based on impressions of original data and its predictions.
* Libraries such as pandas, plotly, statsmodels, and matplotlib.
* The dataset has three columns; Date, Clicks, Impressions
* The model identifies null values, remove unindentified registrations, streamline some data inputs, conduct A/B tests, make visualizations, and predict future CTR.

# Notes
* Although a specific data set is used for this model, it can be easily altered to cater for other datasets.
* Data was acquired from www.statso.io
